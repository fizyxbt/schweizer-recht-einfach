# Zitierweise Schweizer Recht

Basisplugin für Schweizer Quellenarbeit. Es erzwingt SR/AS/BBl, BGE, nicht amtlich publizierte Bundesgerichtsurteile und kantonale Quellen mit Rechtsstand und Fundstelle.
